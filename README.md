# Winter moth local adaptation across Europe pipeline  

This repository contains the computational pipeline used in **Estandia, Learmonth et al.** to explore population structure, selection and adaptation to environmental variables in winter moths (*Operophtera brumata*) from Europe.  
The workflow processes raw sequencing data through variant calling, filtering, population genomic summary statistics, genotype–environment association analyses (LFMM), and estimation of effective migration surfaces. 


---

## Overview  

This pipeline takes raw Illumina reads and performs sequential steps leading to population genetic and landscape genomic inference.
